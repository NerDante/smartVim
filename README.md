## Overview
my vim config, use vim-plug to manage plugins
![image](https://github.com/NerDante/smartVim/screenshot/startPage.png)

## Required
- vim verison > 8.0 (asynchronous completion)
- gtags
- python (leaderF)
- clang (deoplete)

## Install
```
git clone https://github.com/NerDante/smartVim.git
cd smartVim
./install.sh
```
